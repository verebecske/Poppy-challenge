# Compiling the versions

## RUST
use `rustc` to compile `poppy.rs` :
    `rustc poppy.rs`

---
## OCAML
use `ocamlc` to compile `poppy.ml` :
    `ocamlc poppy.ml`

---
## COMMON LISP
use `clisp` to compile `poppy.lisp` :
    `clisp poppy.list`

---
## ELIXIR
use `elixir` to compile and run `poppy.ex`
    `elixir poppy.ex`
